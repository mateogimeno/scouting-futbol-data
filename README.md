# scouting-futbol-data
# Proyecto de Scouting con Python ⚽📊

¡Hola! Soy analista de datos interesado en el deporte. En este repositorio guardo mis herramientas para extraer y procesar datos futbolísticos.

### ¿Qué hace este código?
* **Scraping:** Extraigo datos de rendimiento de ligas desde FBref.
* **Procesamiento:** Limpieza de datos con la librería Pandas.
* **Objetivo:** Identificar talentos sub-23 basándome en métricas avanzadas.

### Herramientas utilizadas:
* Python 3.x
* Pandas / BeautifulSoup
* Datos de: FBref / StatsBomb
